- TodoRepository는 JPARepository를 상속 받은 interface로 구현
- TodoContorller에는 TodoList 조회, 작성, 수정, 삭제 기능 구현
- h2.database 와 연동 JSON 형식으로 API 제공
- AWS Ubuntu 20.01 에 git clone 후 Gradle로 Build
